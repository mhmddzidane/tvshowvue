# TVSHOWHUN

Website mencari informasi mengenai tv show dibuat menggunakan VUE JS & Tailwind CSS

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Link Deployment

https://tvshowvue.vercel.app/
